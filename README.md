# Crashy Planes
This is a simple Flappy Bird clone featuring a raccoon trying to collect trash. The game is over if the player fails to
collect a piece of trash or they make contact with the edge of the screen or a pipe.

![Image](screenshot.png)

## Controls
* Spacebar - Boost upwards a little bit
* F5 - Restart game
* F11 - Toggle fullscreen
* ESC - Quit

## Configuration
The graphics can be configured by modifying config.ini.
